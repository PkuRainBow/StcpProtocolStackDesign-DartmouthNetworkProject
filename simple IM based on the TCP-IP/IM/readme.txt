【1】打开到指定路径
【2】在一个console中，运行服务器：
     make Server
     ./Server
【3】在其他console中运行客户端：
     make Client
     ./Client
【4】按照客户端的 ISTRUCTOR INFO 中的指示输入不同的命令即可
【5】make clean 即可清除生成的目标文件和可执行文件

注意 ：
若重启后客户端连接不上服务器！这属于正常情况，很多同学都遇到了！
推荐的解决方法是：
更改头文件中的端口号；然后重新编译运行；
